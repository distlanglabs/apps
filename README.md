# distlang apps

This repository contains standalone Distlang example apps.

## Install Distlang

Linux/macOS:

```bash
curl -fsSL https://distlang.com/install | bash
```

Windows PowerShell:

```powershell
irm https://distlang.com/install.ps1 | iex
```

Verify install:

```bash
distlang --help
```

## Apps

- `echo/`

## Quick start

```bash
make verify
```
